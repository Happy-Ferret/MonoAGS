    Enum RoomTransitionState

    The room transition state.

    Namespace: AGS.API
    Assembly: AGS.API.dll
    Syntax
    public enum RoomTransitionState

    Fields

    Name Description
    AfterEnteringRoom

    We're almost finished with the room transition. We entered the new room, giving the transition one last opportunity to do some magic.
    BeforeLeavingRoom

    A room transition has started, we haven't left the old room yet.
    InTransition

    We're currently in transition. We have the 2 screenshots, which can be manipulated by the room transition.
    NotInTransition

    We're not currently in a room transition (we're in a normal rendering loop).
    PreparingNewRoomRendering

    We left the old room, we need to prepare the render instructions for the new room before taking the screenshot to be used by the transition.
    PreparingTransition

    We're preparing the transition screenshots (we need to take a screenshot of the old room, then move the player to the new room, set up the camera and take another screenshot).
