    Interface ISquare

    Represents a square.

    Note: the points are listed as bottom left, bottom right, etc, even though it's not accurate. A square doesn't really have bottom left, it just has 4 points. We use these descriptors for convenience, since each square in our engine was converted from a rectangle which does have bottom left, etc. Also, the order of the points is important for the "Contains" calculation.
